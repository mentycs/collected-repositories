STOP! Critical Checkpoint

Before executing, ask yourself these essential questions

🤔
Could I Complete This?
With this plan and context, would I personally be able to complete the task?
  • ✓ Are requirements crystal clear?
  • ✓ Have I referenced the correct part of the codebase?
  • ✓ Have I provided all necessary context?
  • ✓ Are architecture decisions documented?
📊
How Will I Verify Success?
Using these validation techniques, would I be certain of success?
  • ✓ Will tests prove functionality?
  • ✓ Do linters ensure code quality?
  • ✓ Can integration tests verify the system?
  • ✓ Are success metrics measurable?
